Felix's Node.js Guide

이 문서에 대하여.

저는 Felix Geisendörfer 입니다. node.js 초기부터 핵심 contributor로 참여해 왔으며 transloadit.com의 공동설립자이기도 합니다.

지난 몇 달간 사람들과 많은 대화를 나누고 node.js를 사용하여 컨설팅도 많이 했습니다.

나 스스로 같은 일을 계속 반복하는 것 같아서 이 가이드를 써서 node.js를 처음 시작하는 사람들을 돕기로 했습니다. 그래서 휴가를 조금 할애하여 이 일에 투자했습니다.

Content

Node.js Beginner Guide
만약 node.js를 처음 접한다면 이 글을 읽어 주세요. hello world 어플리케이션을 만들고 디플로이할 수 있게 도와줄 거에요.
Node.js Style Guide
node.js에서도 일반적인 자바스크립트 스타일을 추천합니다. 개인에 취향에 따라 많은 방법이 존재하지만 좀 더 일관적인 방법을 알아볼 거에요.
Node.js Community Guide
node.js 커뮤니티들에 대해 알어볼 거에요. 그들이 어디서 노는지 봅시다.
Node.js Convincing The Boss Guide
node.js가 어디에 쓰이는 물건인지 득이 되도록 관리하는 방법에 대해 알아봅시다.

다른 가이드도 준비중입니다. GitHub 레파지토리을 팔로우하면 새로운 가이드를 바로 보실 수 있습니다.

Node.js Package Guide
node.js 패키지를 만들고 npm에 등록publish하는 방법.
Node.js Object Oriented Programing Guide
내가 node.js에서 객체 지향적으로 개발하는 방법.
Node.js Test Driven Development Guide
node.js로 test driven development를 시작하는 방법.
